### 本地仓库与远程仓库绑定

首选进入想要存放项目的本地仓库文件夹（没有就新建一个）

使用 **git** 命令初始化文件夹

```git
git init 
```

然后进行远程连接

```git
git remote add origin https://github.com/xxxxlc/Group-of-ML.git
```
